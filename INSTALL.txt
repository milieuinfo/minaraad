Minaraad

    This is the Plone Product for Minaraad. To install this product make sure   
    all dependencies are installed first and then install it using the quick  
    installer.

Dependencies

    * PasswordResetTool (https://svn.plone.org/svn/collective/PasswordResetTool/trunk)
    
    * Five 1.2 (http://codespeak.net/z3/five/release/Five-1.2.tgz)

    * CompoundField (https://svn.plone.org/svn/archetypes/MoreFieldsAndWidgets/CompoundField/trunk/CompoundField)

    * FCKeditor 2.2 (http://prdownloads.sourceforge.net/fckeditor/FCKeditor.Plone2.2.zip?download)

    * RichDocument2.0rc1 (http://plone.org/products/richdocument/releases/2.0/RichDocument-2.0rc1.tar.gz)
